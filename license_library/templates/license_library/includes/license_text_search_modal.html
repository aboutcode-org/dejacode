License text search
The license text search provides an efficient way to look for a text string in the DejaCode license texts (in contrast to the main search field, which searches license metadata, the fields that explain a license).
  • It is not necessary to enclose the entire text string in quotes.
  • For best results, enter (or paste) a text string that seems to be relatively uncommon; for example, general purpose desktop.
  • It is also recommended that you enter a text string that is characteristic of a particular group of licenses; for example, for any purpose and without fee.
  • There is no limit to the amount of search text you can enter, but you may find it best to start with no more than a paragraph of license text.
