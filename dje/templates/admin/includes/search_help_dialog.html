You can refine your search to operate on specific data fields, using the convention

[field name][operator][search value]

1. Field name. Position your pointer over the search field to get a list of available field names.

2. Operators.
Use = for an exact match
Use ^ for "starts with".
Use : for "contains".

3. Note that the search value is not case sensitive. "apache" and "Apache" and "APACHE" are equivalent values for search purposes.

Examples:
name=apache
will find all cases where the name exactly matches apache or Apache or APACHE.

name^apache
will find all cases where the name starts with apache or Apache or APACHE.

name:apache
will find all cases where the name contains apache or Apache or APACHE.

If you enter a search value without using the special syntax, the application will simply search multiple fields to find those that contain the value you entered.
